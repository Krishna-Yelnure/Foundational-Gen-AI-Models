# Frontier AI Model Comparison (Feb 2026)

This repository contains a structured comparison of 10 frontier AI models as of February 2026.

## What this dataset includes
- Publicly disclosed benchmark results
- Model architecture characteristics
- Access models (open, closed, API)
- Disclosure patterns

## What this dataset does NOT do
- Rank models
- Claim absolute performance
- Predict future scaling trends

## Important notes
- Missing values represent unavailable or undisclosed information.
- Benchmark scores should not be treated as definitive measures of real-world capability.
- Agent capabilities often reflect product design choices rather than raw model intelligence.

## Intended use
Exploratory analysis, research, and discussion around AI evaluation practices.
